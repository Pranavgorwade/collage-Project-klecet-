## ✨ Features

- Student Authentication
- Admin Dashboard
- Department Management
- Faculty Directory
- Events & News
- Placement Section
- Contact Form
- Responsive Design

## 🛠 Tech Stack

Frontend: React, Vite, Tailwind CSS, Axios, Framer Motion

Backend: Node.js, Express.js, MongoDB, Mongoose, JWT

## 🚀 Installation

```bash
git clone https://github.com/Pranavgorwade/collage-Project-klecet-.git
cd collage-Project-klecet-
```

Backend:

```bash
cd backend
npm install
npm run dev
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```
